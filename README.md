# Social Media TypeScript
-----------

## Description

1) using typegoose to write schema of project
2) use bcrypt to hash password
3) use jwonwebtoken to crate token for ez login
4) for test use jest

## Installation

```bash
pnpm i
```

## Running

```bash
pnpm dev
```

## Testing

```bash
pnpm test
```

### You will also see any lint errors in the console.
