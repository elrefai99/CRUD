# Social Media TypeScript
-----------

## Description

1) use Typegoose as MongoDB Schema
        like: 
              ```
                class user{
                  @prop({type: String, required: true}) // MongoDB Types
                  Username: string // Typescript interface
                }
              ```

## Installation

```bash
pnpm i
```

## Running

```bash
pnpm dev
```

## Testing

```bash
pnpm test
```

### You will also see any lint errors in the console.
